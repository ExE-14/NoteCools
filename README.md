# Σ(▼□▼ﾒ) NoteCools

Простое приложение для заметок с выделением приоритета, реализованное на JavaFX.  
Проект создан в первую очередь для практики работы с Java и FXML.

### Оно позволит вам качественно структурировать свои мысли, и поможет не забыть ничего важного.
---

## 🧩 Суть приложения

- Визуальный интерфейс на JavaFX (FXML)
- Установка приоритета заметки, по цветам 
- Сохранение заметок локально
- Фильтрация по приоритету
